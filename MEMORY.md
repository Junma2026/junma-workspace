### 核心业务公式（必须记住）
- **药房进销存推损益：** `期初 + 盘点后发药 + 购药 - 发药 - 期末 = 损益`
  - `期初` = 上年末盘点
  - `盘点后发药` = 年末盘点后到年底的发药
  - `购药` = 本年采购量
  - `发药` = 本年销售量
  - `期末` = 本年末盘点
  - `损益` = 正=盘盈/ 负=盘亏
